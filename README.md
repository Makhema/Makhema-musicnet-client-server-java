# Makhema-musicnet-client-server-java

📌 Overview

MUSICNET is a client-server application developed in Java that allows multiple clients to connect to a central server and interact with shared resources such as music documents and files.

This project demonstrates key concepts in:

Socket Programming
Multithreading
File Handling
GUI Development (JavaFX/Swing)
